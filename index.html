WebSCI

Running example

WebSCI is a hand-coded port of ScummVM's SCI engine to Javascript. Below is an example of the Space Quest 3 non-interactive demo running in a browser. Requires a modern browser with support for the canvas tag.

This example requires a browser that supports the HTML5 <canvas> feature.

Note: Currently missing font rendering, sound output, text parser and a bunch of other stuff. Rendering order also needs work.

Source code based upon the rather excellent ScummVM project. Space Quest is the property of Activision Blizzard, copyright of Space Quest 3 demo used under fair use.
